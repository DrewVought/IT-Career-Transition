# IT Career Transition Project

Transitioning from paramedic/EMS into IT systems administration and security through hands-on labs, with the goal of a full-time career move in mid-to-late fall 2027 (flexible — transition happens when the right offer is in hand).

This repo documents that process: real lab work, real session logs, real troubleshooting — turned into portfolio-quality documentation and resume material.

## Standard

All work governed by [`governing-document.md`](./governing-document.md). Current status tracked in [`PROGRESS.md`](./PROGRESS.md).

## Lab Environment

- **Vought-VM** — Ubuntu Server 25.04, Oracle VirtualBox on Asus M16 (2023). Remote access via Tailscale + SSH.
- **Dedicated lab host (planned ~Module 5)** — Used SFF desktop running Proxmox, network-segmented from home.
